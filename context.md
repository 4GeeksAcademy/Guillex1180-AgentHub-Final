# AgentHub - Contexto del Proyecto

## Descripción General
AgentHub es una plataforma SaaS emergente que permite a las empresas alquilar "agentes de IA". Estos agentes son asistentes inteligentes preconfigurados que pueden equiparse con diferentes "skills" (habilidades como navegar por la web, leer documentos o gestionar calendarios) para ejecutar tareas de negocio específicas.

## Objetivo del Proyecto
El equipo fundador necesita un panel de administración interno (Admin Dashboard). Como desarrollador frontend freelance, la misión es construir un prototipo HTML completamente diseñado e interactivo. Este prototipo servirá para que el equipo valide la interfaz y los flujos de usuario antes de entregarlo al equipo de backend para su integración.

## Metodología
El proyecto sigue un enfoque **Vision to spec**: se deben definir exhaustivamente las reglas y la estructura antes de escribir código. El prototipo actúa como el contrato visual y funcional.

## Alcance de los Datos
Para esta fase de prototipado, no se requiere integración con APIs ni bases de datos reales. Toda la información mostrada en el panel estará "hardcodeada" de manera realista y debe ser consistente en todas las vistas (ej. un agente que aparece en "Gestión de agentes" también debe aparecer en los "Logs de errores").
